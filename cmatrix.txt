Installing	[] cmatrix
_____________________________

You can get cmatrix form a git repository
So you need to install git first, follow "git.txt" in repository.

After installing git.
_____________________________
Type the command :
1. "git clone https://github.com/abishekvashok/cmatrix"
2. "cd cmatrix"
3. "less README.md"

From there read instructions for install.

		OR

Type the command : "sudo apt install tree"

Enter your password, note the password is not visible, just type and press Enter.
The package will get loaded, Please continue by 'y' and Enter.

Your package will install now automatically.

Run cmatrix
______________________________
Type the command : "cmatrix" to run the screen-saver.
