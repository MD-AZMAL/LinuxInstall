Installing	[] wallgen
_____________________________
Follow the steps
_____________________________
Type the commands :

You need to first install git, follow "git.txt" in the repository.

1. "git clone https://github.com/SubhrajitPrusty/wallgen.git"
2. "cd wallgen"

You also need to have python installed, If you have NOT installed python then follow "python.txt" from repository.
3. "pip install --editable  ."
4. "pip3 install --editable  ."

Enter your password if asked anywhere, note the password is not visible, just type and press Enter.
The package will get loaded, Please continue by 'y' and Enter.

Your package is installed now.
